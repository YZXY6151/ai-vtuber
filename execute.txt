AI-VTuber 项目部署说明（跨设备使用指南）
===========================================

📁 项目地址：
https://github.com/YZXY6151/ai-vtuber

本项目为基于本地模型与 Electron 桌面端的 AI 虚拟主播系统，包含 ASR（语音识别）、NLP（大语言模型）、TTS（语音合成）、Electron 桌面 UI 等模块，支持跨平台本地部署。

------------------------------------------------------------
✅ 环境要求
------------------------------------------------------------
1. Git
2. Python 3.10+
3. pip
4. Node.js + npm
5. 可选：Ollama（用于本地运行 LLM 模型，如 LLaMA3）
6. 推荐：VSCode 或其他代码编辑器

------------------------------------------------------------
✅ 克隆项目
------------------------------------------------------------
git clone https://github.com/YZXY6151/ai-vtuber.git
cd ai-vtuber

------------------------------------------------------------
✅ 创建 Python 虚拟环境并激活
------------------------------------------------------------
# Linux/macOS
python -m venv .venv
source .venv/bin/activate

# Windows
python -m venv .venv
.venv\Scripts\activate

------------------------------------------------------------
✅ 安装 Python 依赖
------------------------------------------------------------
pip install -r requirements.txt

> 如没有 `requirements.txt` 可先在原本电脑运行：
> pip freeze > requirements.txt

------------------------------------------------------------
✅ 安装前端依赖
------------------------------------------------------------
cd frontend
npm install

------------------------------------------------------------
✅ 启动各 AI 模块服务
------------------------------------------------------------

# 终端1：启动 ASR 服务
cd services/asr
uvicorn app:app --host 0.0.0.0 --port 8181

# 终端2：启动 NLP 服务（如 Ollama）

uvicorn services.nlp.app:app --host 0.0.0.0 --port 8182


# 终端3：启动 TTS 服务
cd services/tts
uvicorn app:app --host 0.0.0.0 --port 8183

------------------------------------------------------------
✅ 启动前端界面
------------------------------------------------------------

# Web 前端（React/Vite）
cd frontend
npm run dev

# 或 Electron 桌面端（如果配置了）
npm run electron-dev

------------------------------------------------------------
✅ 可选：部署 Ollama 本地模型（仅 NLP 使用）
------------------------------------------------------------

# 安装 Ollama
curl -fsSL https://ollama.com/install.sh | sh

# 拉取模型（如 llama3）
ollama run llama3

------------------------------------------------------------
📦 推荐包含文件（以供他人部署）
------------------------------------------------------------

- README.txt / README.md（部署说明）
- requirements.txt
- .gitignore（已排除模型、venv、缓存）
- .env.example（如有配置项）
- package.json（包含 Electron/Web 启动命令）

------------------------------------------------------------
🔚 完成！
现在可在任意设备上复现完整 AI-VTuber 桌面体验。
------------------------------------------------------------

npm install @vitejs/plugin-react --save-dev
